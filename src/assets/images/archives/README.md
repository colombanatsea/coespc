# Archives presse/bulletins anciens

Ce dossier contient les coupures de presse et bulletins paroissiaux intégrés dans `/histoire.html`.

## Fichiers attendus

| Fichier | Source | Utilisation |
|---|---|---|
| `1953-bulletin-paroissial-kermesse.jpg` | Bulletin paroissial *Le Vieux Clocher* (1953) | Section « Origines 1948-1953 » |
| `1953-la-fete-au-village-coupure-presse.jpg` | Coupure de presse locale des années 1950 | Section « Années 1950 » |

## Format recommandé

- **Type** : JPG ou PNG
- **Largeur max** : 1200 px
- **Poids** : < 400 Ko
- **Orientation** : portrait privilégié (la lecture descendante est naturelle pour une coupure de presse)

## Comment ajouter une image

Simplement : glisser-déposer les 2 fichiers dans ce dossier avec les noms exacts ci-dessus, puis `git commit + push`. Le site les affichera automatiquement dans la page Histoire.
